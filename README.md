# Library Management System (C++ OOP)

## Description
A file-based library record management system implemented in C++ using OOP principles.

## Features
- Add, display, search, update, delete books
- Sort books by title
- Export records to CSV
- File-based storage

## Technologies
- C++
- OOP
- File handling

## How to run
1. Open project in Visual Studio
2. Build & Run (Ctrl + F5)

## Files
- books.txt – storage
- books.csv – exported data
